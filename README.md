# SWG-game
Snake, Water, Gun game designed withh CSS animations and JS power.

Unleash your ultimate power by choosing an item mightier than the CPU's. You will have three options: Snake(S), Water(W) & Gun(G).
Snake can drink water, Water can make the gun sink and The gun can kill the snake.Simple And Easy.
options in circle are given below, chose any of them and select "COMPETE" this will let the cpu chose. After three seconds result will appear on the screen.You can try it as many times.
